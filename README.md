# Portfolio - Ignacio Lo Sasso 🚀

Bienvenido al repositorio de mi portafolio personal. Soy Contador Público (UBA) especializado en Análisis de Datos y Control de Gestión.

## 💻 Sobre el Proyecto

Este portafolio web está diseñado para mostrar mis habilidades técnicas, mi trayectoria profesional y los proyectos en los que he trabajado, combinando mi experiencia en finanzas con el mundo de los datos y el desarrollo.

### Tecnologías Utilizadas
- **HTML5 & CSS3**
- **JavaScript (ES6+)**
- **Tailwind CSS** (para un diseño responsivo y modo oscuro)
- **Formspree** (para el formulario de contacto funcional)

## 🌟 Características
- 🌓 Modo Claro / Oscuro dinámico.
- 📱 Diseño 100% responsivo (Mobile First).
- ✉️ Formulario de contacto integrado.
- 💼 Sección de proyectos renderizada dinámicamente.
